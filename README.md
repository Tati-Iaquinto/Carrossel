# Carrossel

## Dependencias 
    - http-server
## Instruções
    1. Clone o repositorio
        $ git clone link-do-repo
    2. Instale as dependencias
        $ cd nomedorepo
        $ npm install
    3. Rode o projeto
        $ npm run dev
        - Acesse no browser http://localhost:8080/
