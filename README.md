# Carrossel

## Dependencias 
    - http-server
## Instruções
    1. Clone o repositorio
        $ git clone https://github.com/Tati-Iaquinto/Carrossel.git
    2. Instale as dependencias
        $ cd Carrossel
        $ npm install
    3. Rode o projeto
        $ npm run dev
        - Acesse no browser http://localhost:8080/
